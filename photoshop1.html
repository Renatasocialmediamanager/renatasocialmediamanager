Digital Composition: Pizza Project

Project Overview

This project was developed using Adobe Photoshop to create a high-end commercial visual. I focused on the following technical aspects:

  • Layer Masking: To seamlessly integrate the food elements with the background.
  • Color Correction: Adjusting vibrance and saturation to make the product look appetizing.
  • Lighting/Shadows: Creating custom drop shadows to give the composition depth and a 3D feel.
